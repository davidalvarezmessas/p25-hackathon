# p25-hackathon

## SIMULATION D'UN ECO-SYSTEME 

Au cours de la journée du hackathon, nous avons codé une simulation dun éco-système. 

Pour ce faire, nous avons utilisé la programmation orientée objet (POO) avec l'utilisation de plusieurs classes : 
- classe Mouton
- classe Loup 
- classe Herbe
- classe Grid


## Installation
Pour installer le projet (si utile?): 

```bash
pip install -r requirements.txt
```

## Struture du projet 

project/
│── main.py
│── README.md

## Fonctionnalités 

L'utilisateur peut jouer sur la simulation en changeant certains paramètres : 
- Nombre intial des moutons et des loups
- Energie gagnée par les moutons et les loups grâce à l'herbe et aux moutons
- Probabilité de pousse de l'herbe
- Temps de repousse de l'herbe

Il est ainsi possible de construire un éco-système qui soit plus ou moins enclins aux moutons ou aux loups.
